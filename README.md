У нас есть "компьютер" (класс Сomputer) но в нем совсем нет полезных программ.
Создай "программу", которая будет уметь выполнять сложение двух целых чисел и выводить результат в консоль (можешь ее назвать Calculator).
Создай класс Calculator
Создай в нем два метода:
1-й метод должен реализовывать сложения чисел (можешь назвать его getSum)
2-й метод должен выводить результат сложения в консоль (можешь назвать его printResult)
Передачу чисел в класс Calculator можешь реализовать с помощью конструктора или используя сеттеры.
